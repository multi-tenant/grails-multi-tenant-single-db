Interface Summary
CurrentTenant Each tenant is identified by a number.
CurrentTenantAware Beans implementing this interface will get the 'currentTenant' bean injected.
CurrentTenantAwareRepository Current tenant aware repository
MultiTenantDomainClass All multi tenant domain classes should implement this interface.
Tenant Most applications will have one domain class implementing this interface and the tenantId method will most likely return the database generated id to allow for fast lookups from second level cache.
TenantRepository Most applications will probably provide an implementation of this interface.
